1.通过props向子组件传递数据 （父到子）

```
在子组件中定义
props:['son1','son2']
或者使用对象(可以使用类型的限制/默认值)
props:{
son1:Number,
son2:Number
}

在父组件中定义数据
data:{
father1:1,
father2:2
}

使用子组件时，使用父组件中的数据
一定要使用v-bind属性绑定
<cpn :son1='father1' :son2='father2'></cpn>

子组件的模板
<template>
<div>
<h3>
{{son1}}
</h3>
</div>
</template>
```



2.通过事件向父组件发送消息 （子到父）

```
子组件模板：
<template>
<div>
<h3 @click='sonfun'><h3>
</div>
</template>


使用子组件时，绑定事件
<cpn @fun1='fafun'></cpn>


子组件：
methods:{
sonfun(){
console.log('子组件的事件')
//发射一个事件fun1，且把123传给这个事件
this.$emit('fun1',123)
//123会传入到下面的data中
}
}

父组件：
methods:{
fafun(){
console.log('父组件的事件'+data)
}
}
```
