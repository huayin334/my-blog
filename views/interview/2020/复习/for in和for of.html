for in和for of

huayin 2020-04-22 20:02:00 js

# for...in循环出的是key

# for...of循环出的是value

for of是es6新加的，而且不能用来循环遍历普通对象

所以循环对象建议使用for..in，循环数组建议使用for..of

  var obj = {
    hhh:'1234',
    hisadh:9237103
  }
  for (let k in obj) {
    console.log(k);//key
    console.log(obj[k])//value
  }
